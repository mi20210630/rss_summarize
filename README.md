## 2025-07-16 快讯｜Firefox 即将撤离中国

- **背景**  
  Mozilla 与中国运营方“谋智火狐信息技术公司”结束合作，中国版 Firefox 账户体系将随之关闭。

- **影响简述**  
  1. 涉及所有 `@firefox.com.cn` 及其他中国区管辖区 UID。  
  2. 浏览器继续可用，但「同步、附加组件账号、主题、书签」等需迁移到国外主站 (`accounts.firefox.com`)。  
  3. 本地数据需在 **2025-08-15** 前导出（官方已发布备份指南）。  

- **用户行动建议**  
  1. 立即执行：设置 → Firefox 账户 → 同步 → 手动「导出书签/密码」(CSV/HTML)。  
  2. 注册全球账户 → 重新登录 → 上传数据。  
  3. 如在企业网络，确认代理或国际网络可用性。

- **后续进展**  
  Mozilla 支援团队将在 WeChat 公众号「MozillaCN」持续答疑。

> 公告原文经短暂撤回，官方确认“技术公告提前触发”，最终细则会在 **48 小时** 内再次推送。
