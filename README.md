## 今日蓝点网要点  
- **火狐中国版或停服**：官方未回应，但用户需提前准备迁移。  
- **中国版 vs 国际版关键差异**：  
  - 账号体系、扩展商店与隐私政策均不互通。  
- **迁移三步骤**（需手动完成，无自动迁移）：  
  1. 判断版本：地址栏输入 `about:support` →“ distribuitionID ”值为 `Mozilla/Online` 即为中国版。  
  2. 中国版数据备份：  
     - `about:profiles` → 打开根目录 → 备份文件夹：  
       - `bookmarkbackups`（书签）  
       - `storage`（本地存储）  
     - `登录与密码` → `导出为 CSV`。  
  3. 安装国际版 → 新建配置文件 → 将备份文件覆盖对应目录 → 在设置中导入密码 CSV。  
- **注意事项**：  
  - 需重新注册国际版帐号，同步后历史记录与标签页可恢复。  
  - 中国版账号及其已有云数据无法直接迁移，务必备份本地文件。
